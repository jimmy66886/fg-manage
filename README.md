**有食做后台管理系统**

主要功能：

1. 用户管理
   - 禁用/启用用户
2. 分类管理
   - 大类以及分类的增删改查
3. 菜谱管理
   - 新增菜谱数量
4. 轮播图管理
   - 首页轮播图修改
